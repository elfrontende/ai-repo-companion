# Summary

Every durable fact should live in a small note with explicit links and a narrow scope.

# Signals

- One note should capture one concept, one decision, or one recurring pattern.
- Links matter as much as content because retrieval should navigate the graph.
- Notes should be cheaper to retrieve than any wide project document.
